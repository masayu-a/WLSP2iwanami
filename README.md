# WLSP2iwanami

## Description
『分類語彙表』増補改訂版データベースと『岩波国語辞典第五版タグ付きコーパス2004』の語義の対応をとったもの

## Features 

1行目がヘッダです。

各列は以下の通りです。

- 岩波国語辞典sense@senseid
- 岩波国語辞典sense@type
- 分類語彙表情報
- 分類語彙表情報（を見よ参照など）
